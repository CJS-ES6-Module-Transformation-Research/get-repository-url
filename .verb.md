## Usage

```js
var repositoryUrl = require('{%= name %}');

repositoryUrl('generate', function(err, url) {
  console.log(url);
  //=> 'https://github.com/generate/generate'
});
```

**Promise**

If no callback is passed, a promise will be returned:

```js
repositoryURL('generate')
  .then(function(url) {
    console.log(url);
    //=> 'https://github.com/generate/generate'
  });
```
